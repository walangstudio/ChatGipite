# WhatsMyName data attribution

`wmn-data.json` in this directory is a verbatim, unmodified vendored copy of the
WhatsMyName project's data file.

ChatGipite uses this dataset only as detection signatures (it queries each
target platform directly); the file is redistributed unmodified. Any modified
copy of this dataset must remain under CC BY-SA 4.0.
